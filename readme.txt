Displays a tab on the side of the screen which slides out to reveal additional content when clicked.

== Description ==
Add a floating tab to pages on your site which shows additional content (like an ad for your latest product, a contact form, etc) when the user clicks on it. Customize the colors, add images, and add a button link.
